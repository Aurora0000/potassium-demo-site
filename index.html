Rapid Development

No need to wait months for that one feature you've been hoping for - Iodine gets new features all the time.

All your favourite syntactic sugar

Iodine feels natural to most people; if you've ever used Python, Ruby, PHP, or even C#, you'll see similarities and quickly pick up Iodine.

Powerful libraries

The Iodine ecosystem is growing by the minute, and accessing packages couldn't be easier. With Ion , it's simply a case of one command, and you're done!

Friendly community

Iodine's developers and users are a friendly bunch - they'll be happy to help you if you've got a problem or a bug.

Convinced?

Take a look at the simple "Hello, world" code in Iodine. Pretty simple, right? Already, you should start to spot features that you're used to:

  • String Interpolation, like in C# and Python.
  • A print function, like Python.
  • A main function, like in C#

Still not clicked download?

You're getting tough to convince. Perhaps taking a look at Iodine's powerful object-oriented programming features will help?

At first, you quickly notice the self keyword, which Pythonistas should be familiar with. If you're not familiar with it, self is used to make the function a class method, as opposed to a static method. Then, you can access the class through self (just like C#'s this keyword).

You can also see the super() function, which is used to call the parent class' constructor (which shares the name of the class, like in C#). Even if the parent class has no constructor, you still need super (), or else the class won't actually inherit.

Did we mention that semi-colons are optional? If you prefer to omit them, Iodine has no trouble with that. Cool, eh?
